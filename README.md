# Requisitos:

## Priridade Alta
 - Cadastro de novos pacientes; Informações básicas para a simulção de alerta, como prioridade e medicação.
 - Tela principal;
    - Mostra enfermeiros e pacientes com as informações principais.
    - Espaço visual para as configurações e outras ferramentas.
    - Espaço simulando a pulseira.
 - Simulação de alerta; Crítico, como pressão alta, e remédio.
 - Cadastro de novos enfermeiros; Max quatro enfermeiros, com informações básicas.

## Prioridade Média
 - Login de enfermeiro; CPF e senha.
 - Adição ou edição de informações dos pacientes.
 - Confirmção de chamada; outros enfermeiros sabem quando outro atendeu uma chamada.
 - Histórico do paciente; nome da medicação, histórico de doenças e da doença atual, etc.
 - Icones com cores especificas.
